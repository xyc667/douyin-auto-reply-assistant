# 📬 抖音私信功能使用指南

---

## 🚀 快速开始

### 1️⃣ 检查 Cookie

首先运行检查脚本，确保 Cookie 是完整的：

```bash
cd d:\下载\DouYin_Spider-master\DouYin_Spider-master
python check_cookie.py
```

**注意**：必须包含 `sessionid` 和 `s_v_web_id` 字段！

---

### 2️⃣ 运行私信接收程序

```bash
python dy_apis/douyin_recv_msg.py
```

---

## 💬 使用方法

### 接收私信：
- 程序会自动连接抖音私信 WebSocket
- 收到的私信会自动显示在终端

### 回复私信：
- 收到私信后，在终端输入框中输入回复内容
- 按 Enter 键发送
- 程序会自动回复给最后一个发消息的人

### 退出程序：
- 输入 `quit`、`exit` 或 `退出` 并按 Enter
- 或按 `Ctrl + C`

## ⚠️ 常见问题

### Q: 提示 "需要 Cookie 中包含 sessionid 字段"？
A: 请重新获取完整的 Cookie，确保包含 `sessionid` 字段。

### Q: 收到消息但无法回复？
A: 检查是否有对话信息保存，确保 Cookie 有效。

### Q: 程序启动失败？
A: 先运行 `python check_cookie.py` 检查 Cookie。

---

## 📋 检查 Cookie 字段

完整的 Cookie 应该包含以下字段（至少）：
- ✅ `sessionid` - **必需**，用于私信功能
- ✅ `s_v_web_id` - **必需**，用于 API 请求
- ✅ `passport_csrf_token`
- ✅ `msToken`
- ✅ `ttwid`
