# 抖音爬虫使用说明

## 📋 前置准备

### 1. 安装依赖
```bash
pip install -r requirements.txt
npm install
```

### 2. 配置 Cookie

**获取 Cookie 的方法：**
1. 打开浏览器，访问 https://www.douyin.com 并登录
2. 按 F12 打开开发者工具
3. 切换到「网络」(Network) 标签
4. 刷新页面，点击任意请求
5. 在「请求标头」中找到 `Cookie` 字段，复制完整内容

**配置到 .env 文件：**
```env
DY_COOKIES=你的完整Cookie内容
DY_LIVE_COOKIES=
```
注意：不要加引号！

## 🚀 功能使用

### 1️⃣ 数据爬取 (main.py)

#### 功能：
- 爬取指定视频/作品信息
- 爬取用户所有作品
- 搜索视频并批量爬取

#### 使用方法：
```bash
cd 项目目录
python main.py
```

#### 自定义配置：
编辑 `main.py` 文件，取消注释或修改对应功能：
```python
# 1. 爬取指定视频（需替换为有效的视频URL）
works = ['https://www.douyin.com/video/视频ID']
data_spider.spider_some_work(auth, works, base_path, 'all', 'test')

# 2. 爬取用户所有作品
user_url = 'https://www.douyin.com/user/用户ID'
data_spider.spider_user_all_work(auth, user_url, base_path, 'all')

# 3. 搜索视频
query = "搜索关键词"
require_num = 20
data_spider.spider_some_search_work(auth, query, require_num, base_path, 'all', ...)
```

#### 保存选项：
- `all` - 保存所有信息（媒体文件 + Excel）
- `media` - 只保存媒体文件
- `excel` - 只保存 Excel 文件

---

### 2️⃣ 私信收发 (dy_apis/douyin_recv_msg.py)

#### 功能：
- WebSocket 实时接收私信（文本、表情、语音、图片、分享视频）
- 支持自动重连

#### 使用方法：
```bash
cd 项目目录
python dy_apis/douyin_recv_msg.py
```

#### 注意事项：
- 需要 Cookie 中包含 `sessionid` 字段
- 程序运行后会持续监听私信
- 按 `Ctrl+C` 停止

---

### 3️⃣ 直播间监听 (dy_live/server.py)

#### 功能：
- 监听弹幕消息
- 监听礼物（含送礼对象）
- 监听进场/关注/点赞
- 监听房间热度
- 支持直播间发送弹幕、点赞

#### 使用方法：
```bash
cd 项目目录
python dy_live/server.py
```

## ⚠️ 注意事项

1. **Cookie 会过期**：如果提示"请先登录"，需要重新获取 Cookie
2. **反爬机制**：抖音有验证码，频繁请求可能触发验证
3. **仅供学习**：请勿用于商业用途或非法用途
4. **遵守规则**：请遵守抖音平台规则

---

## 🔧 故障排除

### 问题：提示 "请先登录"
**解决**：重新获取最新的 Cookie

### 问题：KeyError: 's_v_web_id'
**解决**：确认 Cookie 完整，包含所需字段

### 问题：ModuleNotFoundError
**解决**：确认已运行 `pip install -r requirements.txt`
